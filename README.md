# C++ Chat Room Server

这是一个用C++编写的简单聊天室服务器程序，支持多用户通过浏览器进行实时聊天。该程序基于Winsock库，使用HTTP协议进行通信。

## 功能特点

- 多用户同时在线聊天
- 消息实时显示在网页上
- 最多保留100条聊天记录
- 支持GET和POST请求处理
- 简洁的HTML界面

## 技术栈

- C++11
- Winsock库
- 多线程
- HTTP协议

## 环境要求

- Windows操作系统
- Visual Studio 2019或更高版本
- C++11支持
- Winsock库
